## 工程简介

本博客系统分为前端管理和后台管理两部分。

前台管理功能包括文章分页显示、文章详情查看、文章评论管理；

后台管理功能包括数据展示、文章发布、文章修改、文章删除、评论管理、文章分类/标签管理，用户管理；

同时有着用户登录功能，对用户进行权限隔离。
